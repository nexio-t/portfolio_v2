Projects

COVID-19 Testing Map

The U.S. Covid Testing Map tracks COVID-19 testing across U.S. States and territories with data from The Covid Tracking Project, a volunteer-driven effort to collect the latest testing figures, led by The Atlantic. Because they only display the data in tabular form on their site, I set out to map it using Leaflet, MapBox, and the Google Maps geocoding API. The data is refreshed once daily by The Covid Tracking Project team.

Parky

Need parking in Philly? Enter Parky, the airBnb for parking spots. With Parky, users may create parking listings, search for spaces by address, reserve spaces, and update or cancel reservations across a range of days in Philly.

My contributions to this project included integrating the Google Maps APIs, dynamic rendering of available parking spaces, database set up, and implementing routing. I was also responsible for creating the app's mockups on Adobe XD (viewable here), which guided our design.
